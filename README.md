# Bulk Deletion Jobs Migrator

An [XrmToolBox plugin](https://www.xrmtoolbox.com/plugins/RVG.XrmToolBox.BulkDeleteMigrator/) designed to seamlessly migrate/copy Recurring bulk deletion jobs from one environment to another, eliminating the need for manual recreation.

## 📦 Installation

1. Open **XrmToolBox**.
2. Go to the **Tool Library**.
3. Search for **BulkDeleteMigrator**.
4. Click **Install**.

## 🛠️ How to Use

1. Open the Bulk Delete Migrator tool in XrmToolbox.
2. Click on **Load Bulk Deletion Jobs** button to fetch all the recurring bulk deletion jobs in the source environment *(If you aren't connected to a source environment, you will be prompted to select one)*.
3. Use the checkboxes to choose the jobs you wish to migrate.
4. Select the Target Environment and click the **Migrate Selected Jobs** button to transfer the selected jobs to the target environment
5. Check the **Logs** section at the bottom for detailed feedback on the migration
